IMPLEMENTATION SUMMARY
======================

PRIORITY 1 (Very Common in Interviews)
======================================

1. Heavy-Light Decomposition
   Python: /python/advanced_ds/heavy_light_decomposition.py
   Java:   /java/advanced_ds/HeavyLightDecomposition.java
   
   Time: O(log² n) path queries on trees
   Use: Tree path min/max/sum queries
   Key: Decompose tree into heavy paths, use segment tree

2. Segment Tree with Lazy Propagation
   Python: /python/advanced_ds/segment_tree_lazy.py
   Java:   /java/advanced_ds/SegmentTreeLazy.java
   
   Time: O(log n) range updates and queries
   Use: Range add, range sum; range assign, range min/max
   Key: Lazy value propagation avoids redundant updates

3. Splay Tree
   Python: /python/advanced_ds/splay_tree.py
   Java:   /java/advanced_ds/SplayTree.java
   
   Time: O(log n) amortized insert/delete/search
   Use: Self-adjusting BST, cache-like behavior
   Key: Splaying recently accessed nodes to root

PRIORITY 2 (Common in String/Pattern Problems)
===============================================

4. AC Automaton (Aho-Corasick)
   Python: /python/advanced_ds/ac_automaton.py
   Java:   /java/advanced_ds/AhoCorasickAutomaton.java
   
   Time: O(n + z) text search, z = matches
   Use: Multi-pattern matching, substring search
   Key: Trie + KMP-style failure links

5. Suffix Array with LCP
   Python: /python/advanced_ds/suffix_array.py
   Java:   /java/advanced_ds/SuffixArray.java
   
   Time: O(n log n) construction, O(m log n) search
   Use: Pattern matching, LRS, space-efficient alternative to suffix tree
   Key: Sorted suffixes + LCP array (Kasai's algorithm)

PRIORITY 3 (Medium Difficulty)
==============================

6. Persistent Segment Tree
   Python: /python/advanced_ds/persistent_segment_tree.py
   Java:   /java/advanced_ds/PersistentSegmentTree.java
   
   Time: O(log n) per version, O((n+m) log n) space
   Use: Version history, undo/redo, historical queries
   Key: Structural sharing reuses unmodified subtrees

7. Dancing Links (DLX)
   Python: /python/advanced_ds/dancing_links.py
   Java:   /java/advanced_ds/DancingLinks.java
   
   Time: O(n!) worst, much better in practice
   Use: Exact cover, Sudoku, N-queens, polyomino tiling
   Key: Circular doubly-linked lists for O(1) removal/restoration

8. KD-Tree
   Python: /python/advanced_ds/kd_tree.py
   Java:   /java/advanced_ds/KDTree.java
   
   Time: O(log n) avg nearest neighbor, O(sqrt(n) + k) range search
   Use: Spatial indexing, k-NN, collision detection
   Key: Recursive space partitioning on alternating dimensions

PRIORITY 4 (Specialized)
========================

9. Van Emde Boas Tree
   Python: /python/advanced_ds/van_emde_boas_tree.py
   Java:   /java/advanced_ds/VanEmdoBoasTree.java
   
   Time: O(log log U) operations on universe of size U
   Use: Integer operations on small universe, successor/predecessor
   Key: Recursive universe partitioning into sqrt(U) blocks

10. Leftist Heap
    Python: /python/advanced_ds/leftist_heap.py
    Java:   /java/advanced_ds/LeftistHeap.java
    
    Time: O(log n) merge, insert, delete-min
    Use: Mergeable priority queue, heap merge operations
    Key: Left path >= right path guarantees log n height

11. Fibonacci Heap
    Python: /python/advanced_ds/fibonacci_heap.py
    Java:   /java/advanced_ds/FibonacciHeap.java
    
    Time: O(1) amortized insert/decrease-key, O(log n) delete-min
    Use: Dijkstra/Prim on sparse graphs, dynamic algorithms
    Key: Forest of Fibonacci trees, cascading cuts

12. B+ Tree
    Python: /python/advanced_ds/b_plus_tree.py
    Java:   /java/advanced_ds/BPlusTree.java
    
    Time: O(log n) search/insert/delete, O(log n + k) range scan
    Use: Database indexing, efficient range queries
    Key: All keys in leaves, internal nodes guide search

13. Cartesian Tree
    Python: /python/advanced_ds/cartesian_tree.py
    Java:   /java/advanced_ds/CartesianTree.java
    
    Time: O(n) construction, O(log n) RMQ queries
    Use: RMQ, nearest smaller element, optimal offline RMQ
    Key: BST on values + heap property on indices

14. Link-Cut Tree
    Python: /python/advanced_ds/link_cut_tree.py
    Java:   /java/advanced_ds/LinkCutTree.java
    
    Time: O(log n) amortized link/cut/path operations
    Use: Dynamic connectivity, tree path queries
    Key: Splay trees for preferred path decomposition

15. Suffix Tree
    Python: /python/advanced_ds/suffix_tree.py
    Java:   /java/advanced_ds/SuffixTree.java
    
    Time: O(n) construction (Ukkonen), O(m) pattern search
    Use: Pattern matching, LCS, longest repeated substring
    Key: Compact trie of suffixes, each edge is substring
