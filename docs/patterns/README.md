# Problem-to-Pattern Matcher

Welcome to the interview prep problem matcher. Use this guide to recognize common problem types and immediately identify the best data structures and algorithms to use.

## How to Use

1. **Identify your problem type** — Start with a category below that matches your problem
2. **Find the pattern** — Look up the specific problem type in that category
3. **Get the approach** — See which data structures and algorithms fit
4. **Reference code** — Check the linked code in this repo for implementation examples
